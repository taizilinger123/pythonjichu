3. Git 

3.1 git clone 
3.2 git status 查看本地库的状态
3.3 git branch 分支名 创建分支
3.4 git branch -d 分支名 删除分支
3.5 git checkout 分支名 切换分支
3.6 git checkout -b 分支名 新建并切换分支
3.7 git add  提交文件到暂存区
3.8 git commit 提交到本地代码库(记得加-m注释)
3.9 git push origin 本地分支:远程分支
    git push origin kdh:kdh 
3.10 merge合并
3.11 git pull origin 远程分支
3.12 git tag -a 表签名 -m '注释'
     git tag -a v0.1 -m "0.1"
     git tag
     git push origin kdh:kdh --tags
